# Cách chạy
sau khi clone về
cd đến folder trealet

```
cd trealet
```
sau đó chạy lệnh 

```
php -S 127.0.0.1:8000

```
sau đó truy cập:

http://127.0.0.1:8000/albums/Nhom15/app/map xem phần show-off

truy cập 

http://127.0.0.1:8000/albums/Nhom15/app/admin để  xem phần tạo file json
